# English
# chemist_try
It includes an agent that can select a chemical element, and then the user can ask questions to narrow down the answer. The AI can only respond with yes or no, ultimately guessing the answer.

## Download
<a href="https://github.com/Frank8020151/chemist_try/releases/tag/chemist_try_v0.3.0_Beta">Click to download</a>

## Quick Start
Fill in your **api_key** and **base_api** in the **Main.py** and **tools.py** files, then run **Chemist_try.py** to use.

**Model requirements:**

1. Support for tool invocation

2. Text generation

## Exception Handling:
Unable to connect to API: Check if your API_KEY is valid  
Cannot connect: Possibly no internet connection
